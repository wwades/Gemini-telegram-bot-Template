# Gemini Telegram Bot Template

A simple and clean template for building a Telegram bot powered by **Google Gemini AI** using Python. This repository is designed to be a "plug-and-play" base for your own AI projects.

---

## Features
* **Gemini Pro Integration:** Uses Google's latest generative model for intelligent responses.
* **Asynchronous:** Built with `python-telegram-bot` for fast, non-blocking performance.
* **Environment Safety:** Uses `.env` files to keep your API keys secure and private.
* **Clean & Documented:** Easy to read and extend for your own custom logic.

---

## Quick Start

### 1. Get Your API Keys
You will need two keys to run the bot:
1.  **Telegram Bot Token:** Get it from [@BotFather](https://t.me/BotFather) on Telegram.
2.  **Gemini API Key:** Get it for free at [Google AI Studio](https://aistudio.google.com/).
